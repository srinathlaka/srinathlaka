## 🚀 Featured Projects

### 1. Local AI Agent for CI/CD Log Analysis

A local AI-assisted troubleshooting tool that analyzes CI/CD logs and generates root-cause recommendations.

**Highlights**
- Parses CI/CD pipeline logs
- Detects and classifies failure types
- Generates severity and confidence scores
- Uses a local LLM through Ollama
- Provides troubleshooting explanations
- Includes Streamlit UI, file upload, history, and report export
- Includes GitHub Actions syntax checks

**Tech stack:** `Python` · `Streamlit` · `Pandas` · `Requests` · `Ollama` · `qwen2.5` · `GitHub Actions`
